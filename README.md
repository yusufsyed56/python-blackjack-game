# Blackjack Python Game ♠️

A terminal-based Blackjack game built using Python.

## Features
- Play against the computer dealer
- Random card generation
- Blackjack score calculation
- Win/Lose/Draw conditions
- Simple interactive terminal gameplay

## Concepts Used
- Functions
- Lists
- Loops
- Conditional Statements
- Random Module
- Game Logic

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Open the project folder
3. Run:
